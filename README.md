# Tesla Range Calculator


## Technology stack
CSS3, ES6, ReactJs, JSX

## Browser
Chrome

## Responsive
No

## Project URL
[Link](https://tesla-range-calculator-hy.herokuapp.com/)

## Install in localhost
1. Install npm
```
brew install node
```

2. Install all dependencies
```
npm install 
```

3. Run project and enjoy
```
npm run start
```

4. The web will run at 
```
http://localhost:3000
```
